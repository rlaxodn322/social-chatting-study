**React**: 19.0.0  <br>
**Tailwind CSS**: 3.4.1  <br>

**Axios**: 1.7.9  <br>

### 구동 방법
**포트 번호**: `3000`  <br>
아래 명령어를 실행하여 필요한 패키지를 설치 후 개발 서버 실행  <br>
```sh
npm i
npm run dev
```
